HTML & CSS Quick Reference Guide

A short guide on how to use HTML and CSS

HTML tags and elements

  • <h1> - <h6>

    • Headers give a number of headings, starting with the largest (<h1>) to the smallest (<h6>) to smallest.

    <h1>This is an example of a heading</h1>
  • <p>

    • The paragraph tag is used to create paragraphs in webpages

    <p>This is a paragraph.</p>
  • <a>

    • The <a> tag is used to create a hyperlink by using a link address 

    <a href="https://www.google.com/">Google</a>
  • <img>

    • The <img> tag is used to embed an image in an HTML page 

    <img src="flower.jpg" alt="flower">
  • <ul>, <ol>, <li>

    • Defines unordered or ordered lists and list items.

    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
    <ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>
  • <table>

    • Creates tables for organizing data.

    <table>
    <tr>
        <td>Table Data</td>
    </tr>
</table>
  • <audio>

    • The <audio> tag is used to embed sound content in a webpage

    <audio controls>
    <source src="music.mp3" type="music/mp3">                 
</audio>
  • <video>

    • The <video> tag is used to embed video content in a webpage

    <video controls>
    <source src="video.mp4" type="video/mp4">
</video>
  • <u>

    • The <u> tag is used to underline text
  • <b>

    • The <b> tag is used to bolden a text.

    Properties of a CSS

    • color: Sets the color of the text.
    • font-family: Sets the font style of a text.
    • background-image: Sets a background image in the webpage.
    • font-size:This property sets the size of the text.
    • margin:This allows to create space around elements, outside of any defined borders .
    • border: The border property allows to specify the style, width, and color of an element's border .
    • text-align:Allows to align text horizontally (center,left,right).
    • height and width:Used to set the height and width of an element .
    • position:property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).
